# v1.0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** 实现文件/图片上传问答、WebDAV 数据备份、关于页面三个功能。

**Architecture:**
- 文件上传：前端读取文件为 base64，通过 `SendMessageRequest.Attachments` 传给后端，后端构造 eino multipart message 发给 LLM（仅支持 vision 模型）
- WebDAV 备份：后端用 `golang.org/x/net/webdav` 客户端（实际用 `studio.storied.space/gowebdav` 或手写 HTTP），将 `~/.wails-chat/chat.db` 上传/下载到 WebDAV 服务器
- 关于页面：设置弹窗新增 `about` tab，显示版本号（编译时注入）、简介、GitHub 链接

**Tech Stack:** Go + eino schema.ImageURL / schema.FileURL, gowebdav, Vue 3

- [ ] **Step 2: 构造多模态 eino 消息**

在 `StreamChat` 里，构建 user message 时，如果有附件则用 `schema.MultiContent`：

```go
// internal/handler/chat.go — buildUserMessage 辅助函数
func buildUserMessage(content string, attachments []Attachment) *schema.Message {
    if len(attachments) == 0 {
        return &schema.Message{Role: schema.User, Content: content}
    }
    parts := []schema.ChatMessagePart{}
    if content != "" {
        parts = append(parts, schema.ChatMessagePart{
            Type: schema.ChatMessagePartTypeText,
            Text: content,
        })
    }
    for _, a := range attachments {
        if strings.HasPrefix(a.MimeType, "image/") {
            parts = append(parts, schema.ChatMessagePart{
                Type: schema.ChatMessagePartTypeImageURL,
                ImageURL: &schema.ChatMessageImageURL{
                    URL:    "data:" + a.MimeType + ";base64," + a.Data,
                    Detail: "auto",
                },
            })
        } else {
            // 文本文件：直接附加内容到文本
            decoded, err := base64.StdEncoding.DecodeString(a.Data)
            if err == nil {
                parts = append(parts, schema.ChatMessagePart{
                    Type: schema.ChatMessagePartTypeText,
                    Text: fmt.Sprintf("\n\n[文件: %s]\n%s", a.Name, string(decoded)),
                })
            }
        }
    }
    return &schema.Message{Role: schema.User, MultiContent: parts}
}
```

在 `IgnoreContext` 和普通历史两个分支里，把最后一条 user 消息替换为 `buildUserMessage(req.Content, req.Attachments)`。

- [ ] **Step 2: 创建 BackupHandler**

`internal/handler/backup.go`：

```go
package handler

import (
    "fmt"
    "io"
    "os"
    "path/filepath"
    "time"

    "github.com/studio-b12/gowebdav"
    "light-ai/internal/storage"
)

type BackupHandler struct{}

func NewBackupHandler() *BackupHandler { return &BackupHandler{} }

type WebDAVConfig struct {
    URL      string `json:"url"`
    Username string `json:"username"`
    Password string `json:"password"`
    Path     string `json:"path"` // remote path, e.g. /Light/
}

func getWebDAVConfig() (*WebDAVConfig, error) {
    url, _ := storage.GetSetting("webdav_url")
    if url == "" {
        return nil, fmt.Errorf("WebDAV 未配置")
    }
    username, _ := storage.GetSetting("webdav_username")
    password, _ := storage.GetSetting("webdav_password")
    path, _ := storage.GetSetting("webdav_path")
    if path == "" { path = "/Light/" }
    return &WebDAVConfig{URL: url, Username: username, Password: password, Path: path}, nil
}

func (h *BackupHandler) SaveConfig(url, username, password, path string) error {
    storage.SetSetting("webdav_url", url)
    storage.SetSetting("webdav_username", username)
    storage.SetSetting("webdav_password", password)
    storage.SetSetting("webdav_path", path)
    return nil
}

func (h *BackupHandler) GetConfig() (WebDAVConfig, error) {
    cfg, err := getWebDAVConfig()
    if err != nil {
        return WebDAVConfig{Path: "/Light/"}, nil
    }
    cfg.Password = "" // 不返回密码到前端
    return *cfg, nil
}

func (h *BackupHandler) Backup() error {
    cfg, err := getWebDAVConfig()
    if err != nil { return err }

    home, _ := os.UserHomeDir()
    dbPath := filepath.Join(home, ".wails-chat", "chat.db")

    f, err := os.Open(dbPath)
    if err != nil { return fmt.Errorf("打开数据库失败: %w", err) }
    defer f.Close()

    client := gowebdav.NewClient(cfg.URL, cfg.Username, cfg.Password)
    client.Connect()

    remotePath := cfg.Path + "chat-" + time.Now().Format("20060102-150405") + ".db"
    if err := client.WriteStream(remotePath, f, 0644); err != nil {
        return fmt.Errorf("上传失败: %w", err)
    }
    return nil
}

func (h *BackupHandler) ListBackups() ([]string, error) {
    cfg, err := getWebDAVConfig()
    if err != nil { return nil, err }

    client := gowebdav.NewClient(cfg.URL, cfg.Username, cfg.Password)
    client.Connect()

    files, err := client.ReadDir(cfg.Path)
    if err != nil { return nil, fmt.Errorf("读取目录失败: %w", err) }

    var names []string
    for _, f := range files {
        if !f.IsDir() {
            names = append(names, f.Name())
        }
    }
    return names, nil
}

func (h *BackupHandler) Restore(filename string) error {
    cfg, err := getWebDAVConfig()
    if err != nil { return err }

    client := gowebdav.NewClient(cfg.URL, cfg.Username, cfg.Password)
    client.Connect()

    remotePath := cfg.Path + filename
    reader, err := client.ReadStream(remotePath)
    if err != nil { return fmt.Errorf("下载失败: %w", err) }
    defer reader.Close()

    home, _ := os.UserHomeDir()
    dbPath := filepath.Join(home, ".wails-chat", "chat.db")
    // 先备份当前数据库
    os.Rename(dbPath, dbPath+".bak")

    f, err := os.Create(dbPath)
    if err != nil { return fmt.Errorf("创建文件失败: %w", err) }
    defer f.Close()

    if _, err := io.Copy(f, reader); err != nil {
        os.Rename(dbPath+".bak", dbPath) // 回滚
        return fmt.Errorf("写入失败: %w", err)
    }
    return nil
}
```

## 注意事项

1. **文件上传大小限制**：base64 传输大文件会很慢，建议前端限制单文件 ≤ 10MB，并在 UI 上提示。
2. **Vision 模型限制**：图片问答只有支持 vision 的模型才能用（gpt-4o、claude-3 等），普通模型会报错，需要在 UI 上给出友好提示。
3. **WebDAV 恢复需重启**：恢复数据库后需要重启 app 才能生效，需要在 UI 上提示用户。
4. **密码安全**：WebDAV 密码存在 SQLite 明文，`GetConfig` 接口不返回密码到前端，只在后端使用。
